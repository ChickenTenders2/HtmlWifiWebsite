Technical Details of Wi-Fi

Wireless Standard

Wireless Standard defines a set of specifications and protocols that represent how wireless devices
communicate with each other over radio frequences. These standards are established by standardization organizations
such as the Institute of Electrical and Electronics Engineers (IEEE) and the Wi-Fi Alliance, to ensure compatibility
across devices from different manufacturers/brands. Wireless Standards set the rules for transmitting data over
radio frequencies,such as frequency bands, data trasmission, and security protocols.

Frequency Bands

A frequency band, also referred to as a band, is a range of frequencies in the electromagnetic frequency spectrum that are designated for particular uses in
the field of telecommunications. The very low frequencies (VLF) and very high frequencies (THF) make up the division of this spectrum. There are upper and
lower frequency limits for each frequency band. The range or gap of radio frequencies used to send a signal over a telecom network is known as a
frequency band. Every band, and there are a lot of them, is distinguished by its lower and upper bounds. These different ranges of frequencies are all
a part of the electromagnetic spectrum. Frequency bands are used to regulate the energy transmitted by electromagnetic waves.

  • View "History" page for the different versions of Frequency Bands throughout the years.

Data Trasmission

WiFi uses radio waves to transfer data. Data is transformed into a radio signal by a device's wireless adapter, which then transmits it via an antenna.
This procedure works both ways, the router receives data from the Internet, transforms it into a radio signal, and then transmits the signal back to
the device for decoding. Through this procedure, users of the web can download and upload content from the Internet; in fact, sending URLs through your
browser is considered two-way communication.

Security Protocols

Wi-Fi security protocols uses encryption technology to safeguard networks and client data. Wireless security methods are essential for
keeping you safe online because wireless networks are frequently less secure than wired ones. Data encryption tools protect sensitive
data by scrambling it until it's unreadable. Cryptographic keys are used by Wi-Fi security protocols to randomise data and render it
unreadable. Since Wi-Fi systems use symmetrical encryption, the same key is used to encrypt and decrypt data.

All Wi-Fi security protocols are certified by the Wi-Fi Alliance, the non-profit organization that owns the Wi-Fi trademark.

At the moment, there are four wireless security protocols available:

  • Wired Equivalent Privacy (WEP)
  • Wi-Fi Protected Access (WPA)
  • Wi-Fi Protected Access 2 (WPA 2)
  • Wi-Fi Protected Access 3 (WPA 3)
